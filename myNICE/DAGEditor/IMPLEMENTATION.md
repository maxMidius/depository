| Method | Endpoint | Purpose |
|--------|----------|---------|
| GET | `/api/dags` | List all DAGs |
| GET | `/api/dags/{name}` | Load specific DAG |
| POST | `/api/dags/save` | Save a DAG |
| DELETE | `/api/dags/{name}` | Delete a DAG |
| POST | `/api/dags/import` | Import DAG from file |
| POST | `/api/dags/{name}/export` | Export DAG to file |
| GET | `/api/stats` | Get system statistics |
| GET | `/api/health` | Health check |

## 📞 Support

### Common Issues & Solutions

**Q: Port 8000 already in use?**
A: Change port in `app_advanced.py` or kill existing process

**Q: Module not found errors?**
A: Ensure virtual environment is activated and requirements installed

**Q: Cannot write to storage?**
A: Check directory permissions with `chmod 755 storage`

**Q: Need to change port?**
A: Edit line in `app_advanced.py`:
```python
ui.run(..., port=YOUR_PORT)
```
